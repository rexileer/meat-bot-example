# MeatBot (example)

Концепт производственной системы: Telegram‑бот (цех) + Django CRM (офис). Этот пример предназначен для ознакомления со структурой и подходами; работоспособность не гарантируется.

## Кратко о проекте
- Единая доменная модель между ботом и вебом
- Модули: обработчики, состояния, локализация, интеграции
- Веб-часть: Django-панель и CRM-модули

## Структура
- `bot/` — логика Telegram-бота, обработчики, клавиатуры, middlewares
- `Web/` — Django-проект (настройки, CRM, шаблоны, статика)
- `data/` — конфигурация и вспомогательные файлы

## Безопасность
- Все чувствительные данные удалены или заменены placeholder’ами
- Не добавляйте реальные токены/ключи в репозиторий

## Примечание
Это пример (example): запуск/деплой умышленно не описаны. Используйте проект как справочник по архитектуре и структуре.
